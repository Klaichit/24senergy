วิธีทำ: ติดตั้ง WordPress ผ่าน cPanel (1-click) → สั่งทำ custom theme จาก design ที่เราทำไว้ → ใช้ WPML หรือ Polylang สำหรับ TH/EN → ใช้ ACF (Advanced Custom Fields) หรือ Elementor Pro เพื่อให้แก้ section ได้ตามที่ออกแบบไว้
ข้อดี
  • ใช้ Hostatom ที่ซื้อแล้วเต็มที่ — ไม่เสียเปล่า
  • ทีม content แก้เองได้ผ่าน wp-admin (มี admin built-in)
  • WPML รองรับ TH/EN ครบทุก field
  • Plugin มหาศาล: SEO (Yoast), form (CF7), backup
  • หาคนช่วยดูแลในไทยง่ายมาก (WP shop เพียบ)
  • Cost ระยะยาวต่ำ — ของ Hostatom + WPML license
ข้อเสีย
  • Performance ช้ากว่า Next.js (~3-5s LCP vs < 2s)
  • ต้องดูแล security plugin/update WP
  • Custom theme ใหม่ราคา ~15,000-30,000฿ ครั้งเดียว
  • WPML license ~$99/ปี
  • UI/UX อาจไม่ pixel-perfect เท่า static ที่เราทำ
เพิ่มเติม: ~20,000฿ ครั้งเดียว + ~3,500฿/ปี (WPML)

7 ขั้นตอน จาก Hostatom ที่มีอยู่ → เว็บไซต์พร้อมใช้

1
Login cPanel ของ Hostatom → คลิก "WordPress Toolkit" หรือ "Softaculous" → Install WordPress ที่ 24senergy.co.th
2
หา WordPress theme developer ในไทย — ลองหาใน Pantip, Fastwork, หรือ Hostatom เองมี service · ส่ง design HTML ที่เราทำให้เป็น reference
3
ติดตั้ง plugin หลัก: WPML (TH/EN), ACF Pro (custom fields), Yoast SEO, Contact Form 7 (RFQ form), UpdraftPlus (backup)
4
สั่งทำ theme ตาม design — ใช้ structure ที่ออกแบบไว้ (Hero, KPI, Products, Solutions, Projects, Footer) · บอก dev ใช้สี #6A2DAF + #3F3F3F
5
ใช้ ACF สร้าง field editable ตามที่ออกแบบไว้ — ทีม content จะเห็น form แก้แต่ละ section ใน wp-admin คล้ายๆ Sanity
6
Import เนื้อหาเริ่มต้น + รูป จาก HTML ปัจจุบัน → ตั้งใน wp-admin ครบทุกหน้า · เปิด Maintenance Mode ก่อน launch
7
Train ทีม content 1-2 ชม. · ทำคู่มือ (1 หน้า) · ปิด Maintenance → Launch 🚀
