Tips สำหรับแต่ละ Platform

🤖DALL-E 3 / ChatGPT

ตัด --ar --v ออก · พิมพ์เป็นประโยคธรรมดา · บอก aspect ratio ในข้อความ "wide cinematic 16:9 aspect ratio" · DALL-E ไม่ตอบ negative prompt ตรงๆ แต่ตอบจาก context

⚡Flux (Pro / Schnell)

ผ่าน Gemini app หรือ Google Whisk · เก่ง landscape และ aerial · prompt สั้นกระชับได้ผลดีกว่ายาว · มีฟรีให้ใช้ผ่าน gemini.google.com

เคล็ดลับเพิ่มเติม

1. สั่งหลาย variations แล้วเลือกที่ดีที่สุด

Midjourney/Flux มักให้ 4 ภาพต่อ prompt · gen 3-4 รอบเลือกที่ดีที่สุดก่อน upscale

2. ทำ "Hero" + "Detail" สองชุด

Prompt 1 หรือ 3 → ใช้เป็น hero ใหญ่ · Prompt 2 หรือ 5 → ใช้ในการ์ดเล็กหรือ section อื่น · ให้ภาพในเว็บมีหลากหลาย

3. ระวังขนาดไฟล์

หลังได้ภาพ → resize เป็น 1920×1080 px + แปลงเป็น WebP ที่ squoosh.app · ขนาดไม่ควรเกิน 200 KB

4. ภาพควรมี "negative space"

เพราะจะมี text headline ทับ — เลือกภาพที่มีพื้นที่ว่าง 1/3 ของภาพให้ตัวอักษรอ่านง่าย · ถ้าไม่มีก็ใส่ gradient overlay สีเข้มทับได้

5. ใส่ภาพมาให้ผมแล้วบอกตำแหน่ง

พอได้ภาพแล้ว upload เข้ามาในแชท + บอก "ใช้ภาพนี้ใน Hero" — ผมจะนำไปวางใน index.html ปรับ gradient overlay ให้ text อ่านง่าย และ optimize ขนาดให้ครับ
